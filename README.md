# CellTag-Main-Figure-code

## Overview

This repository contains the code used to reproduce Figures 1–5 from the CellTag manuscript.

## Repository structure

Figure1/
Code used to generate Figure 1.

Figure2/
Code used to generate Figure 2.

Figure3/
Code used to generate Figure 3.

Figure4/
Code used to generate Figure 4.

Figure5/
Code used to generate Figure 5.

script/preprocessing/
Preprocessing scripts for scRNA-seq, CellTag, and hashtag analyses.

script/analysis/
Downstream analysis scripts.

## Installation

No installation is required.

Users only need access to an R environment with the required packages installed.

Typical installation time: less than 10 minutes.

## Demo

The scripts provided in the Figure1–Figure5 directories can be used to reproduce the figures presented in the manuscript.

Expected output:

* Figure 1 panels
* Figure 2 panels
* Figure 3 panels
* Figure 4 panels
* Figure 5 panels

Expected runtime:

Approximately 5–30 minutes depending on the computational environment and dataset size.
